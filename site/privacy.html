Конфиденциальность RootVPN

Мы храним только минимум данных, необходимых для работы сервиса:

  • Telegram ID/username (при работе через Telegram);
  • служебные данные доступов и устройств;
  • статусы и идентификаторы платежей.

Эти данные используются только для выдачи доступа, поддержки и стабильной работы сервиса.

Мы не продаем персональные данные третьим лицам.
